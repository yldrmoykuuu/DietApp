# Healty Diyet Uygulaması

Bu proje, Flutter ile geliştirilmiş, kullanıcıların sağlık hedeflerine ulaşmalarını kolaylaştıran bir diyet ve kalori takip asistanıdır.

## ✨ Temel Özellikler

Uygulama, sağlıklı bir yaşam tarzı benimsemenize yardımcı olacak bir dizi güçlü araç sunar:

* **📈 Akıllı Kalori Takibi:**
    * Günlük olarak tüketilen kalorileri kolayca kaydedin.
    * Makro (Protein, Karbonhidrat, Yağ) dağılımını izleyin.
    * Hedeflerinize göre günlük almanız gereken kalori miktarını görün.

* **💧 Su Tüketimi Hatırlatıcısı:**
    * Günlük su hedefinizi belirleyin.
    * Unutmanızı engellemek için akıllı bildirimler ve hatırlatıcılar alın.


* **🔐 Kullanıcı Doğrulama (Giriş & Kayıt):**
    * Güvenli e-posta/şifre ile kayıt olma ve giriş yapma sistemi.
    * Kişisel verileriniz ve ilerlemeniz hesabınızda saklanır.

* **📊 Detaylı Raporlama ve İstatistikler:**
    * Haftalık ve aylık ilerlemenizi (kilo, kalori, su) grafikler üzerinde görün.
    * Geçmiş verilerinizi analiz ederek motivasyonunuzu artırın.

* **🎯 Kişisel Hedef Yönetimi:**
    * Kilo verme, kilo alma veya kilonuzu koruma gibi kişisel hedefler belirleyin.
    * Uygulama, bu hedeflere ulaşmanız için size yol göstersin.

* **👤 Kullanıcı Profili:**
    * Kişisel bilgilerinizi (boy, kilo, yaş, aktivite seviyesi) yönetin.
    * Uygulama ayarlarını kişiselleştirin.

## 🛠️ Kurulum

Projeyi yerel makinenizde çalıştırmak için:

1.  Depoyu klonlayın:
    ```sh
    git clone [https://github.com/SENIN-KULLANICI-ADIN/healty.git](https://github.com/SENIN-KULLANICI-ADIN/healty.git)
    ```
2.  Proje dizinine gidin:
    ```sh
    cd healty
    ```
3.  Gerekli paketleri yükleyin:
    ```sh
    flutter pub get
    ```
4.  Uygulamayı çalıştırın:
    ```sh
    flutter run
    ```

# Healty Diyet Uygulaması

Bu proje, Flutter ile geliştirilmiş bir diyet ve kalori takip uygulamasıdır.

## Özellikler

- Günlük kalori takibi
- Su tüketimi hatırlatıcısı
- Adımsayar entegrasyonu
